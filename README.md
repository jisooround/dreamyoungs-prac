# 꿈많은청년들

기간: 2023.08.01~08

### 실행방법

```
$ npm install
```

```
$ npm run start
```

### 사용 라이브러리

- react-datepicker : 달력 라이브러리
- react router dom : 라우팅
- vite-plugin-mkcert : local환경에서 https로 실행시키기 (vite config에서 https 설정하였습니다.)
